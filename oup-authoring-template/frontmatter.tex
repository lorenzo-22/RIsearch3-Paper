\abstract{%
\textbf{Motivation:} RNA--RNA interaction prediction underpins siRNA design and non-coding RNA analysis. RIsearch2 is a widely used tool but predates modern packaging and the Python data ecosystem, limiting integration into contemporary analysis pipelines.\\
\textbf{Results:} We present RIsearch3, a Rust reimplementation of the RIsearch core with algorithmic refinements, a redesigned command-line interface, and Python bindings. We also rewrite the siRNA off-target pipeline in modern Python, using Polars for tabular processing and YAML for configuration; the pipeline doubles as a reference example of the Python bindings. Benchmarks show a 2--3$\times$ speedup on the core and substantially faster end-to-end pipeline runs.\\
\textbf{Availability and implementation:} RIsearch3 is released under [LICENSE] at \href{https://github.com/}{GitHub URL}. The Rust binary installs via \texttt{cargo install risearch3}; the Python package is available on PyPI as \texttt{risearch3}.\\
