\abstract{%
\textbf{Motivation:} RNA--RNA interaction prediction underpins siRNA design and non-coding RNA analysis. RIsearch2 is a widely used tool but predates modern packaging and the Python data ecosystem, limiting integration into contemporary analysis pipelines.\\
\textbf{Results:} We present RIsearch, a package comprising a Rust reimplementation of the RIsearch2 search algorithm with Python bindings, together with a modernised siRNA off-target pipeline that consumes those bindings directly. The search core reproduces RIsearch2 output while running 2--3$\times$ faster; the pipeline installs with a single \texttt{pip install}, is configured from one YAML file, and completes its post-search stages substantially faster than the legacy Python~2.7 implementation.\\
\textbf{Availability and implementation:} RIsearch is released under [LICENSE] at \href{https://github.com/}{GitHub URL}. The Rust binary installs via \texttt{cargo install risearch}; the Python packages are available on PyPI as \texttt{risearch} (search core) and \texttt{risearch-pipeline} (off-target pipeline).\\
